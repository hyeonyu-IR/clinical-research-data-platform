# AVS Registry Quickstart

1. Launch app:
```bash
streamlit run streamlit_avs_registry_app.py
```
2. Set CSV path in sidebar (default: `data/avs/avs_registry.csv`).
3. Enter one case and click **Save Case**.
4. Confirm the row in **Review / Export** and download CSV backup.
5. Check descriptive charts in **Dashboard**.
